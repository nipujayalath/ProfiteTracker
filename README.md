# Profit Tracker

A modern web application for tracking daily delivery business profits, built with React, Vite, and Node.js/MongoDB.

## Features
- **Dashboard**: Visualize revenue, costs, and net profit trends over time
- **Data History**: View and manage past delivery records
- **Reporting**: Export detailed delivery reports to CSV
- **Custom Time Filtering**: Analyze data daily, monthly, yearly, or custom date ranges.

## Tech Stack
- **Frontend**: React (Vite), Recharts, Lucide-React
- **Backend**: Node.js, Express, Mongoose
- **Database**: MongoDB

## Running Locally

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the application (Backend on 5000, Frontend on 5173):
   ```bash
   npm run dev
   ```
